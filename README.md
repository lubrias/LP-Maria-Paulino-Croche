# Maria Paulino Atelier

Landing page institucional e comercial para a Maria Paulino Atelier, com foco em perfumes, bolsas autorais em crochê, tapetes e peças personalizadas.

## Visão Geral

O projeto foi pensado para conversão no mobile, com visual elegante, paleta inspirada na identidade da marca e CTA direto para WhatsApp.

## Funcionalidades

- Hero com proposta de valor e chamadas para ação.
- Seção de perfumes com exemplares masculino e feminino.
- Vitrine de bolsas autorais com fotos reais.
- Seção editorial com destaque visual de uso da peça.
- Bloco institucional com foto da Maria Paulino.
- CTA fixa de WhatsApp no celular.
- Formulário de contato que abre conversa no WhatsApp com mensagem pronta.
- Layout responsivo e otimizado para performance mobile.

## Tecnologias

- HTML5
- CSS3
- JavaScript puro
- Google Fonts

## Como Executar Localmente

1. Abra a pasta do projeto no VS Code ou no seu editor preferido.
2. Abra o arquivo `index.html` no navegador.
3. Para testar com servidor local, você pode usar a extensão Live Server no VS Code.

## WhatsApp

O contato principal está configurado para abrir conversa via WhatsApp com mensagem pré-preenchida.

Para usar um número real, substitua o placeholder `5500000000000` em:

- `index.html`
- `script.js`

## Publicação no GitHub

O projeto está pronto para publicação em GitHub Pages ou qualquer hospedagem estática.

Sugestão de fluxo:

1. Fazer commit das alterações.
2. Enviar para o repositório remoto.
3. Ativar GitHub Pages na branch `main`.
